# Java Developer Internship – Task 5

## Task
Arrays & Basic Data Analysis Program

---

## Objective
The objective of this task is to understand and implement array operations in Java. 
The program performs basic data analysis such as sum, average, minimum, maximum, sorting, and demonstrates both single-dimensional and multi-dimensional arrays along with exception handling.

---

- Java (OpenJDK 8+)
- IntelliJ IDEA
- GitHub

## Program Features
- Stores numerical data in arrays
- Calculates sum and average
- Finds minimum and maximum values
- Sorts array manually (without built-in methods)
- Demonstrates single-dimensional and multi-dimensional arrays
- Handles ArrayIndexOutOfBoundsException
- Uses loops efficiently
- Displays formatted output
